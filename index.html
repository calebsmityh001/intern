    <!-- Application Structure Plan: The single-page application is structured into distinct thematic sections accessible via a sticky top navigation bar: Welcome, The Day, Your Journey, and Connect. This non-linear, section-based approach allows interns to navigate freely to topics of interest, enhancing explorability over a traditional linear slide deck. 'Welcome' provides initial context. 'The Day' presents the schedule interactively with expandable details for each session and AI-powered summaries. 'Your Journey' focuses on intern value and career growth pathways with interactive cards and AI-generated career insights. 'Connect' offers a dynamic Q&A section for the panel discussion and networking details. This design prioritizes intuitive content grouping and user-driven exploration, making complex information easily digestible and engaging. The 'About Us' section, including its charts and animated stats, has been explicitly removed as per user request. -->

    <!-- Visualization & Content Choices:

        - The Day (Agenda): Utilizes a dynamic vertical timeline (HTML/CSS/JS) to display the day's schedule. Goal: Organize, inform about change, provide details, summarize. Interaction: Current session highlights automatically. Each timeline item is expandable (click to reveal description, presenter, and supporting links) and defaults to being open. A "Summarize Session" button is now present, and AI-generated summaries will only appear upon clicking this button. Justification: Provides a clear, engaging chronological overview while offering on-demand access to rich details and optional AI-powered summaries, giving users control over when summaries are generated. Method: HTML structure with Tailwind for styling, JS for active state, toggle/expansion logic, and Gemini API fetch triggered by button click for summaries. The detailed bullet points from the source are now explicitly rendered as an HTML list within each session's description.

        - Your Journey (Career Paths): Presents potential career paths as interactive cards (HTML/CSS/JS) and includes an AI-powered "Career Path Insights" generator. Goal: Organize, inform, explore. Interaction: Clicking on a card now toggles the visibility of its detailed description, which is visible by default. Users can input a career interest to get AI-generated insights. Justification: Allows interns to explore career details on demand and receive personalized, dynamic insights, encouraging proactive discovery of opportunities. Method: HTML divs with Tailwind for styling, JS for click event listeners to toggle description visibility and Gemini API fetch for career insights. The 'E-commerce Analytics' career path has been re-added.

        - Connect (Q&A Panel Questions): Uses an accordion interface (HTML/CSS/JS) to list the pre-defined questions for the panel. Goal: Inform, prepare for engagement. Interaction: Clicking a question expands to show its text. Justification: Efficiently presents multiple questions in a compact space, allowing interns to review and consider them before the live panel, facilitating a more productive Q&A session. Method: HTML buttons/divs, JS to toggle display property.

        - Supporting Documents: Integrated directly as clickable links within the expanded agenda items for relevant sessions, now correctly parsed from the provided text, including inline and multi-line links. Goal: Provide access to resources. Interaction: Direct click to external URL. Justification: Places context-specific resources directly where they are most relevant in the schedule.

        - General Visuals: Unicode characters/icons are used throughout for visual cues and aesthetic enhancement, adhering to the "no SVG" rule. All charts rendered via Canvas (Chart.js), respecting container sizing.

    -->
